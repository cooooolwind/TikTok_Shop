## 请求体 

- model string 必选 您需要调用的应用的 ID （Bot ID），获取 Bot ID Y 。 

- messages object[] 必选 到目前为止的对话组成的消息列表。不同模型支持不同类型的消息，如文本、图片等。 

## System message 

object 

系统消息，开发人员提供的指令，模型应遵循这些指令。如模型扮演的角色或者目标等。 

- messages.role string 必选 发送消息的角色，此处应为 system 。 

- messages.content string / object[] 必选 系统信息内容。
