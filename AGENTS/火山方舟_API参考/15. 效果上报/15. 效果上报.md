## 15. 效果上报 

## 15.1 方舟 Seedance 2.0 Badcase 上报 API 

方舟Seedance2.0 Badcase 上报 API 

## 使用说明 

当使用Seedance2.0/fast生成视频后，可以通过此 API 把对应的 task_id 上报为 badcase。 

## 注意事项 

未经您的单独同意，火山引擎不会存储和使用您的数据来训练或优化模型。由于Seedance2.0近期需求强 劲，我们针对Seedance2.0系列模型效果类问题提供排障服务，可通过此API提交Badcase至方舟进行排查 分析。如果您通过此方法提交Badcase，则视为您同意使用方舟排障服务，我们将严格按照用户协议章节 3.7.7之（2）进行数据处理。 详细条款请查阅：https://www.volcengine.com/docs/82379/1104498?lang=zh。 本API及文档仅支持中国大陆火山方舟服务，不支持非中国大陆地区以及Byteplus等。 

## 请求说明 

通过 AK SK 调用：请参考 https://www.volcengine.com/docs/6291/216571?lang=zh 获取火山引擎 AKSK， 对应的子用户需开启CasePlatformAccess策略。 

## 使用火山方舟 **API Key** 调用 

除了AKSK，该API支持使用方舟API Key调用，示例如下，方舟API Key需要从 API Key 管理 获取。注意 使用方舟API Key和AKSK的请求方式不同，请求API URL不是 https://open.volcengineapi.com，而是 https://prompt-pilot.cn-beijing.volces.com，另外需要添加路由 /case-platform 。 

bash 

curl --location 'https://prompt-pilot.cn-beijing.volces.com/case-platform? Action=CasePlatformV1BadcaseReport&Version=2024-01-01' \ --header 'accept: application/json, text/plain, */*' \ --header 'content-type: application/json' \ --header'Authorization: Bearer 方舟API_KEY' \ --header 'Region: cn-beijing' \ --header 'ServiceName: ark' \ --header 'Version: 2024-01-01' \ --data-raw '{ "Id": "cgt-20260501132104-tfm9v", "Feedback": "人物手部扭曲，且嘴型和中文配音对不上", "AccountId": "2000000000", "BadcaseType": "character_id_drift", "Source": "huosu", "ContactInfo": "user@example.com" }'
