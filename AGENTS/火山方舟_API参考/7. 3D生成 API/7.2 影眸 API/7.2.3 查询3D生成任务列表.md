## 7.2.3 查询3D生成任务列表 

GET https://ark.cn-beijing.volces.com/api/v3/contents/generations/tasks?page_num={page_num}&page_size={page_size} &filter.status={filter.status}&filter.task_ids={filter.task_ids}&filter.model={filter.model} 

当您要查询符合条件的任务，您可以传入条件筛选参数，返回符合要求的任务。 

## 说明 

仅支持查询最近 7 天的历史数据。时间计算统一采用UTC时间戳，返回的7天历史数据范围以用户 实际发起批量查询请求的时刻为基准（精确到秒），时间戳区间为 [T-7天, T)。

filter.**status ** string 过滤参数，查询某个任务状态。 

- queued ：排队中的任务。 

- • running ：运行中任务。 

- cancelled ：取消的任务，只能查询到 24h 内取消的任务。取消任务超出24h，会被删除。 

- • succeeded ： 成功的任务。 • failed ：失败的任务。 

filter.**task_ids ** string[] 3D 生成任务 ID，精确搜索，支持同时搜索多个任务 ID。多个任务 ID 之间通过 & 连接。 示例：同时搜索多个任务 ID

## usage.total_tokens integer 

本次请求消耗的总 token 数量。3D 生成模型不统计输入 token，输入 token 为 0，故 total_tokens=completion_tokens。
