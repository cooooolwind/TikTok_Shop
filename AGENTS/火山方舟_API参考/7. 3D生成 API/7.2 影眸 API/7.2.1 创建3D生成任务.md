## 7.2.1 创建3D生成任务 

POST https://ark.cn-beijing.volces.com/api/v3/contents/generations/tasks 本文介绍创建影眸 3D 生成任务 API 的输入输出参数，供您使用接口时查阅字段含义。模型会依据传入的 图片信息生成 3D，待生成完成后，您可以按条件查询任务并获取生成的 3D 文件。 

## 说明 

调用本 API 时，按输出 Token 计费，价格为 0.06 元 / 千 Token。 每输出一个 3D 模型固定消耗 3 万 Token（即 1.8 元/次），另有资源包可供选择，详见 模型价格 Y 。 

## 模型支持的 3D 生成能力简介 

- Hyper3d-Gen2 

   - 文生 3D：根据您输入文本提示词 + 参数（可选） 生成一个 3D 模型文件。 

   - ◦ 图生 3D：根据您输入图片（1-5张）+文本提示词（可选）+ 参数（可选） 生成一个 3D 模型文件。

## 文本信息 object 输入给模型的文本内容。 

属性 

content.**type ** string %%require%% 输入内容的类型，此处应为 text 。 

content.**text ** string %%require%% 输入给模型的文本内容，包括： 文本提示词： 仅支持英文，长度不超过 400 字符，超过后将截断。 参数（选填） ：通过 --[parameters] 的方式，控制 3D 文件输出的规格，详情见 模型文本命令(选填） 。 

seed integer 种子整数，用于控制生成内容的随机性。 取值范围：[0, 65535] 之间的整数。 

## 注意 

- 相同的请求下，模型收到不同的 seed 值，将生成不同的结果。 

- 相同的请求下，模型收到相同的 seed 值，会生成类似的结果，但不保证完全一致。 

callback_url string 填写本次生成任务结果的回调通知地址。当 3D 生成任务有状态变化时，方舟将向此地址推送 POST 请 求。 

回调请求内容结构与 查询任务API 的返回体一致。 回调返回的 status 包括以下状态： 

- queued：排队中 

- running：任务运行中 

- succeeded： 任务成功（如发送失败，即5秒内没有接收到成功发送的信息，回调三次） 

- failed：任务失败（如发送失败，即5秒内没有接收到成功发送的信息，回调三次）

"content": [ { "type": "text", "text": "Complete full-body quadrupedal mech robot, orange and black armored, rounded head with glowing blue sensors, 4 articulated mechanical legs, hard-surface sci-fi design, highdetail joints, photorealistic 8K, fully assembled, no missing parts. --mesh_mode Raw --hd_texture true --material PBR --addons HighPack --quality_override 1000000 --use_original_alpha false -- bbox_condition [100,100,100] --TAPose false" } 

] 

material string 默认值 PBR 用于控制 3D 模型的材质类型。可选值： 

- PBR ：物理基础材质，包括基础颜色纹理、金属度纹理、法线纹理和粗糙度纹理，提供高真实感并在 

- 动态光照下具有物理准确性。 

- Shaded ：风格化材质，包括基础颜色纹理和烘焙光照。 

- All ：同时生成 PBR 和 Shaded 材质。 

- None ：无纹理，即生成白模。 

mesh_mode string 默认值 Quad 用于控制生成模型的网格形状。可选值： 

- Raw ：生成三角面模型。 

- • Quad ：生成四边面模型。 

**quality_override ** number 用于自定义 3D 模型的多边形面的面数。 

- 当参数 mesh_mode 为 Raw 时：[500, 1000000]，默认值： 500000 • 当参数 mesh_mode 为 Quad 时：[1000, 200000]，默认值： 18000 

## 说明 

• 参数 quality_override 与 subdivisionlevel 同时使用时，subdivisionlevel 将失效。 • 建议指定面数 ≥150000 面。

addons string 用于增强纹理贴图的分辨率。可选值： 

- HighPack ：提供 4K 分辨率的纹理贴图。 

## 说明 

不配置该参数时，将默认生成 2K 分辨率的纹理贴图。 

use_original_alpha boolean 默认值 false 用于控制 3D 模型的透明度信息。 

- true ：上传图片的透明部分会被保留，生成的 3D 模型或贴图会按原图的透明轮廓处理。 

- • false ：透明部分会被自动填充，可能丢失原始透明效果。 

bbox_condition integer[] 指定模型边界的尺寸和缩放因子的数组。该数组包含三个元素，分别是宽度（x轴），高度（y轴）和长度 （z轴）。如： [100,100,100] 

## 说明 

如无特殊需求，建议不指定该参数，模型将自行判定边界。 

TAPose boolean 默认值 false 生成类人模型时，控制生成的结果是否为T/A Pose。 

- false ：不强制 T/A Pose，由模型自行决定姿态。 

- • true ：强制使用标准绑定姿态，并由模型判定是 T-Pose 还是 A-Pose。 

**subdivisionlevel ** string 简写 sl 控制 3D 模型的多边形面的数量。可选值： 

当 **mesh_mode ** 为 Raw 时，默认值为 high 。 

- high ：500k 

- medium ：150k 

- low ：20k 

当 **mesh_mode ** 为 Quad 时，默认值为 medium 。 

- high ：50k 

- medium ：18k 

- low ：8k

## 7.2.2 查询3D生成任务 

GET https://ark.cn-beijing.volces.com/api/v3/contents/generations/tasks/{id} 查询 3D 生成任务的状态。 

## 说明 

仅支持查询最近 7 天的历史数据。时间计算统一采用 UTC 时间戳，返回的 7 天历史数据范围以用 户实际发起查询请求的时刻为基准（精确到秒），时间戳区间为 [T-7天, T)。
