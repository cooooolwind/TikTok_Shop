## 7.2.4 取消或删除3D生成任务 

DELETE https://ark.cn-beijing.volces.com/api/v3/contents/generations/tasks/{id} 取消排队中的 3D 生成任务，或者删除 3D 生成任务记录。

## 说明 

该参数为 Path Parameter。您需要将待查询的 3D 生成任务的 ID，直接替换 URL 中的 {id} 占位符。 

id string %%require%% 需要取消或者删除的 3D 生成任务 ID。 任务状态不同，调用 DELETE 接口，执行的操作有所不同，具体说明如下：

|---|---|---|---|
|queued|是|任务取消排队，任务<br>状态被变更为<br>cancelled。|cancelled|
|running|不支持|-|-|
|succeeded|是|删除3D生成任务记<br>录，后续将不支持查<br>询。|-|
|failed|是|删除3D生成任务记<br>录，后续将不支持查<br>询。|-|
|cancelled||-|-|

## 7.3 数美 API 

## 7.3.1 创建3D生成任务 

POST https://ark.cn-beijing.volces.com/api/v3/contents/generations/tasks 本文介绍创建数美 3D 生成任务 API 的输入输出参数，供您使用接口时查阅字段含义。模型会依据传入的 图片信息生成 3D，待生成完成后，您可以按条件查询任务并获取生成的 3D 文件。 

## 说明 

调用本 API 时，按输出 Token 计费，价格为 0.08 元 / 千 Token， 每输出一个 3D 模型花费 5 元至 13 元之间（视生成模式而定），详见 模型价格 。 

## 模型支持的 3D 生成能力简介 

- Hitem3d-2.0 

   - 图生 3D：根据您输入图片（ 1-4 张）+文本提示词（可选）+ 参数（可选） 生成一个 3D 模型文件。
