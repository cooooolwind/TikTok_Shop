## 7.3 数美 API 

## 7.3.1 创建3D生成任务 

POST https://ark.cn-beijing.volces.com/api/v3/contents/generations/tasks 本文介绍创建数美 3D 生成任务 API 的输入输出参数，供您使用接口时查阅字段含义。模型会依据传入的 图片信息生成 3D，待生成完成后，您可以按条件查询任务并获取生成的 3D 文件。 

## 说明 

调用本 API 时，按输出 Token 计费，价格为 0.08 元 / 千 Token， 每输出一个 3D 模型花费 5 元至 13 元之间（视生成模式而定），详见 模型价格 。 

## 模型支持的 3D 生成能力简介 

- Hitem3d-2.0 

   - 图生 3D：根据您输入图片（ 1-4 张）+文本提示词（可选）+ 参数（可选） 生成一个 3D 模型文件。
