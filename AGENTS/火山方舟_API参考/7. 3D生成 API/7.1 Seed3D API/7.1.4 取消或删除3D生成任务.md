|failed|是|删除3D生成任务记<br>录，后续将不支持查<br>询。|-|
|cancelled||-|-|

## 7.2 影眸 API 

## 7.2.1 创建3D生成任务 

POST https://ark.cn-beijing.volces.com/api/v3/contents/generations/tasks 本文介绍创建影眸 3D 生成任务 API 的输入输出参数，供您使用接口时查阅字段含义。模型会依据传入的 图片信息生成 3D，待生成完成后，您可以按条件查询任务并获取生成的 3D 文件。 

## 说明 

调用本 API 时，按输出 Token 计费，价格为 0.06 元 / 千 Token。 每输出一个 3D 模型固定消耗 3 万 Token（即 1.8 元/次），另有资源包可供选择，详见 模型价格 Y 。 

## 模型支持的 3D 生成能力简介 

- Hyper3d-Gen2 

   - 文生 3D：根据您输入文本提示词 + 参数（可选） 生成一个 3D 模型文件。 

   - ◦ 图生 3D：根据您输入图片（1-5张）+文本提示词（可选）+ 参数（可选） 生成一个 3D 模型文件。
