## 7.1.1 创建3D生成任务 API 

POST https://ark.cn-beijing.volces.com/api/v3/contents/generations/tasks 运行 Y 本文介绍创建3D生成任务 API 的输入输出参数，供您使用接口时查阅字段含义。模型会依据传入的图片 信息生成3D，待生成完成后，您可以按条件查询任务并获取生成的3D文件。 

模型支持的3D生成能力简介 

- doubao-seed3d 

   - doubao-seed3d-2-0-260328：图生3D，根据您输入图片（1张）+参数（可选） 生成一个带纹理和 PBR 材质的3D文件。
