## 请求参数 

下方仅列出该接口特有的请求参数和部分公共参数。更多信息请见公共参数 。 

Action string %%require%% 示例值：ListFoundationModelVersions 要执行的操作，取值：ListFoundationModelVersions。 

Version string %%require%% 示例值：2024-01-01 API的版本，取值：2024-01-01。 

FoundationModelName string %%require%% 示例值：doubao-seed-2-0-pro 基础模型名称 

PageNumber integer 可选 示例值：1 分页查询时的起始页码，从 1 开始，默认为 1 

PageSize integer 可选 示例值：1 分页查询时每页显示的记录数，取值： 

- 最小值：1 

- 最大值：100 

- 默认值：10 

Filter object 可选 示例值：\待查询基础模型版本的筛选条件 

Description string 可选 基础模型版本描述，模糊匹配 

ModelVersions string[] 可选 基础模型版本名称列表，精准匹配 

Statuses string[] 可选 基础模型版本状态列表，精准匹配，Unpublished / Published / Retiring
