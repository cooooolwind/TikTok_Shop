## 请求参数 

下方仅列出该接口特有的请求参数和部分公共参数。更多信息请见公共参数 G 。 

Action string %%require%% 示例值：GetUsage 要执行的操作，取值：GetUsage。 

Version string %%require%% 示例值：2024-01-01 API的版本，取值：2024-01-01。 

BatchJobId string 可选 示例值：批量推理任务 ID 

Scenes string[] 可选 示例值：Ordinary 场景 

ProjectName string 可选 示例值：default 

- 资源所属的项目名称，默认值为default。 

- 若资源不在默认项目中，需填写正确的项目名称，获取项目名称，请查看文档 G 。 

EndpointIds string[] 可选 示例值：test-ep-id 接入点 id 

StartTime integer %%require%% 示例值：1731945600 开始时间 

EndTime integer %%require%% 示例值：1731945600 结束时间 

Interval integer %%require%% 示例值：3600 查询粒度，按天：86400，按小时：3600
