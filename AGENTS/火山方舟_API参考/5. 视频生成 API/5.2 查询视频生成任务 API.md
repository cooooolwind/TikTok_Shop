## 5.2 查询视频生成任务 API 

GET https://ark.cn-beijing.volces.com/api/v3/contents/generations/tasks/{id} 运行 查询视频生成任务的状态。 

## 说明 

仅支持查询最近 7 天的任务记录，时间区间为 [T-7天, T)，其中 T 为请求发起时刻的 UTC 时间戳 （精确到秒）。注意：视频 URL 有效期为 24 小时，请及时下载或转存。

## 5.3 查询视频生成任务列表 

GET https://ark.cn-beijing.volces.com/api/v3/contents/generations/tasks?page_num={page_num}&page_size={page_size} &filter.status={filter.status}&filter.task_ids={filter.task_ids}&filter.model={filter.model} 运行 GZ 通过传入筛选参数，查询符合条件的视频生成任务。 

## 说明 

仅支持查询最近 7 天的任务记录，时间区间为 [T-7天, T)，其中 T 为请求发起时刻的 UTC 时间戳 （精确到秒）。注意：视频 URL 有效期为 24 小时，请及时下载或转存。

page_num integer / null 默认值：1 返回结果的页码。 取值范围：[1, 500]

**filter.status ** string / null 过滤参数，查询某个任务状态。 

- queued ：排队中的任务。 

- running ：运行中任务。
